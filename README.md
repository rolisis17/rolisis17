## Featured Projects

### TDPN

Trust-Tiered Decentralized Privacy Network MVP scaffold. A Go-based distributed network where one node can run as client, entry relay, exit relay, directory, issuer, or WireGuard bridge. The project explores trust feeds, relay selection, federation, anti-abuse controls, deployment automation, and operational runbooks.

### fastpos

Simplified proof-of-stake blockchain node written in Go. It includes signed transactions, validator voting, mempool controls, persistence, P2P envelopes, Docker testnet configs, product settlement flows, and tests around consensus, persistence, replay protection, and validator lifecycle behavior.

### Bot-AL

Python automation and recognition playground for Albion Online experiments. It combines screenshot analysis, image templates, audio cues, OCR, local dashboards, dataset curation, and iterative detection workflows.

### webserv

HTTP server written from scratch in C++98. It handles configuration parsing, static files, CGI, uploads, custom error pages, request validation, and HTTP response generation without a web framework.

### minishell

Unix shell written in C as a group project. It implements parsing, pipes, redirects, heredocs, environment expansion, built-ins, signal handling, process execution, and Readline history.
